# Nova Scotia Invitational Beach Sprints Regatta — Simple Website

This is a small static website (HTML/CSS/JS) based on Bulletin #1.

## Files
- `index.html` — the page content
- `styles.css` — styling
- `script.js` — mobile nav, countdown, and email helper

## Customize
Search for:
- `info@rownovascotia.ca` (placeholder email)
- dates/times in `script.js` for the countdown (`2026-08-28T09:00:00`)
- accommodation lists and map links in `index.html`

## Publish
You can host these files on any static web host:
- GitHub Pages
- Netlify
- Vercel
- A basic web server

No build step required.
